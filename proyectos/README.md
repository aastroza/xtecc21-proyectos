# Proyectos

Usa esta carpeta para registrar los proyectos finales de XTECC21.

Cuando un equipo quede definido, crea una carpeta con un nombre corto y claro. Dentro de esa carpeta guarda:

- una ficha breve del problema que quieren resolver;
- evidencia o notas de feedback docente;
- avances de la app o prototipo;
- enlaces o instrucciones para probar la solución.

Cada proyecto debe explicar qué actividad cotidiana automatiza, cómo usa Codex y qué queda listo para mostrar en Showcase ExploraTec.
