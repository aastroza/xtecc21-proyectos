# XTECC21 proyectos

Repositorio de trabajo para el proyecto final del curso XTECC21: Crea tu agente de IA.

El objetivo del curso es que cada equipo use Codex para automatizar una actividad real del día a día de estudiantes. La meta es llegar a una app o herramienta usable, idealmente integrada con Codex, que resuelva un problema concreto y pueda presentarse en Showcase ExploraTec.

## Contenido inicial

- `documentos/fuentes/`: documentos originales recibidos para Showcase ExploraTec.
- `notas/`: resúmenes operativos y acuerdos que salen de correos, clases y feedback docente.

## Punto de partida

La primera nota de trabajo está en `notas/showcase-exploratec-2026.md`. Resume fechas, sedes, plazos, documentos para estudiantes y dudas que hay que confirmar antes de enviar material oficial.
